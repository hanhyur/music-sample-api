# music-api
## music-sample-api 설계 및 구현
화면 설계서 기반 api 개발
-> 아티스트 및 앨범, 음원 관리 페이지

### Stack
- Java 8
- Spring Boot
- Maven
- H2

### 기능구현
- Artist
    - 등록
    - 상세 보기
    - 목록 조회 & 검색
    - 수정
    - 삭제
    
- Album
    - 등록
    - 상세 보기
    - 목록 조회 & 검색
    - 수정
    - 삭제
    
- Track
    - 등록
    - 수정
    - 삭제
